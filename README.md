# RentCar Pro

Sistema de locação de veículos desenvolvido com foco em gerenciamento de reservas, clientes, veículos e processos administrativos de uma locadora.

## Sobre o projeto

O **RentCar Pro** é uma aplicação web desenvolvida para simular e organizar o funcionamento de uma locadora de veículos, permitindo controlar cadastros, reservas e informações importantes do sistema de forma centralizada.

O projeto foi criado como parte do processo de aprendizado e prática em desenvolvimento web, unindo conceitos de back-end, front-end, banco de dados e organização de regras de negócio.

## Objetivos

- Gerenciar veículos disponíveis para locação
- Organizar o cadastro de clientes
- Controlar reservas e locações
- Centralizar informações administrativas
- Aplicar conceitos reais de desenvolvimento de sistemas

## Funcionalidades

- Cadastro de clientes
- Cadastro de veículos
- Gerenciamento de reservas
- Controle de disponibilidade de veículos
- Organização de informações da locadora
- Interface administrativa para gestão do sistema

## Tecnologias utilizadas

- PHP
- MySQL
- JavaScript
- HTML5
- CSS3
